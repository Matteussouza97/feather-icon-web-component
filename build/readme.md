# Feather Icons Web Component
A super simple web component for Feather Icons.

## Usage
- Clone or download the repo.
- Include the path to `feather-icon-component.js`
- Slap `<feather-icon> </feather-icon>` wherever you need it.
- Profit.


## Attributes
`width` - in pixels.

`height` - in pixels.

`color`


## Example

```html
<feather-icon
    width="50"
    height="50"
    color="slateblue"
></feather-icon>
```
